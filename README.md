# eslint-plugin-wtf

Catch ridiculous code that should never hit master.

# Usage

Install via npm:

```console
$ npm install --save-dev eslint-plugin-wtf
```

And include it in your eslint config:

```json
{
	"plugins": ["wtf"],
	"extends": ["plugin:wtf/strict"]
}
```
